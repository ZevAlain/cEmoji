# 安装指南

## 📥 下载

### 最新版本下载

访问 [GitHub Releases](https://github.com/ZevAlain/cEmoji/releases) 页面下载最新版本。

![GitHub Releases](https://img.shields.io/github/release/ZevAlain/cEmoji.svg)

## 🖥️ Windows 安装

### 方法 1：直接运行（推荐）

1. 下载 `cEmoji.exe` 文件
2. 解压到任意目录
3. 双击 `cEmoji.exe` 启动应用
4. 首次运行会创建配置文件夹

### 方法 2：管理员运行

如果遇到权限问题：

1. 右键点击 `cEmoji.exe`
2. 选择 "以管理员身份运行"
3. 点击 "是" 确认

## 🔧 系统要求

| 要求 | 最低版本 |
|------|----------|
| Windows | Windows 7 SP1 或更新 |
| Python | 3.8+（如果从源码运行） |
| 内存 | 256 MB |
| 磁盘空间 | 50 MB |

## 📦 从源码安装

### 前置要求

- Python 3.8+
- pip
- git

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/ZevAlain/cEmoji.git
cd cEmoji

# 2. 创建虚拟环境（可选但推荐）
python -m venv venv
venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行应用
python -m cEmoji
```

## 🐛 故障排除

### 问题：应用无法启动

**解决方案：**
```bash
# 检查 Python 版本
python --version

# 重新安装依赖
pip install --upgrade -r requirements.txt
```

### 问题：权限被拒绝

**解决方案：**
- 使用管理员账户运行
- 检查文件夹权限
- 确保有写入磁盘的权限

### 问题：缺少依赖库

**解决方案：**
```bash
# 重新安装所有依赖
pip install -r requirements.txt --force-reinstall
```

## ✅ 验证安装

安装完成后，运行以下命令验证：

```bash
# 检查应用版本
cemoji --version

# 或启动应用
cemoji
```

## 📝 更新应用

### 更新到最新版本

1. 下载最新版本
2. 关闭当前应用
3. 使用新版本文件替换旧版本
4. 重新启动应用

> **注意**：配置文件和已导入的表情包会被保留。
