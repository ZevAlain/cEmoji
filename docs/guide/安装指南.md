# 安装指南

## Windows 版本

1. 打开 [GitHub Releases](https://github.com/ZevAlain/cEmoji/releases/latest)。
2. 下载最新压缩包。
3. 解压到任意目录。
4. 双击外层 `cEmoji.exe` 启动。

当前发布包是 Nuitka standalone 目录版，外层 `cEmoji.exe` 是启动器，实际运行文件和依赖位于 `app` 目录。

## 从源码运行

需要 Python 3.10 或更新版本。

```bash
git clone https://github.com/ZevAlain/cEmoji.git
cd cEmoji
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python cEmoji.py
```

## 更新

1. 关闭正在运行的 cEmoji。
2. 下载并解压新版本。
3. 使用新版本目录启动。

如需保留本地表情，请不要删除旧目录中的 `resource/emoji`、`resource/emoji_small` 和 `etc` 数据。
