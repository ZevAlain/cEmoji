# 功能说明

cEmoji 的核心流程很简单：导入表情，点击复制，在其他软件里粘贴。

## 导入

- 支持 PNG、JPG、JPEG、GIF 图片。
- 支持导入 ZIP 表情包，ZIP 内可以包含子目录。
- 支持从剪贴板读取图片，适合配合 Snipaste、Win+Shift+S 等截图工具使用。
- 导入完成后会显示成功、重复跳过和失败数量。
- 已有相同内容的图片会自动跳过，避免重复表情。

## 使用

- 左键点击表情：复制到剪贴板。
- 在聊天、文档或其他软件中按 Ctrl+V 粘贴。
- GIF 表情会保留动图复制能力。
- 鼠标悬停在 GIF 上 1 秒会播放预览。

## 管理

- 搜索框可以按文件名快速筛选表情。
- 右键表情可置顶、取消置顶或删除。
- 置顶表情会排在前面，并持久保存。
- “管理”模式可快速删除表情。
- “清空所有表情包”会删除本地表情和缩略图。

## 窗口和快捷键

- “始终置顶”可让窗口保持在其他窗口上方，旧版切换置顶闪烁问题已经修复。
- 支持配置全局快捷键，用于呼出或隐藏主界面。
- 最小化会隐藏到系统托盘。
- 程序只允许单实例运行。
