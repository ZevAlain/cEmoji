# 功能说明

## 📋 功能列表

### ✅ 已实现的功能

#### 1. 📦 导入表情包图片

支持导入常见图片格式的表情包：
- PNG 格式
- JPG/JPEG 格式
- BMP 格式
- GIF 格式

**使用方法：**
```
点击 "导入" -> 选择图片文件 -> 点击 "确认导入"
```

#### 2. 📌 当前窗口始终置顶

点击工具栏的 "置顶" 按钮，保持 cEmoji 窗口始终在其他应用之上，便于快速访问。

**功能特点：**
- ✓ 单击启用/禁用
- ✓ 状态指示器
- ✓ 全局快捷键支持（可配置）

#### 3. 📁 导入 ZIP 格式表情包

支持批量导入表情包：

**支持的压缩格式：**
- ZIP 文件（推荐）
- 7z 文件
- RAR 文件

**使用方法：**
```
点击 "导入" -> 选择 ZIP/7z/RAR 文件 -> 自动解压并导入
```

#### 4. 🗑️ 本地表情包可选择删除

灵活管理表情包库：

**功能特点：**
- ✓ 单个删除
- ✓ 批量删除
- ✓ 删除前确认
- ✓ 回收站恢复（如果启用）

**使用方法：**
```
右键点击表情包 -> 选择 "删除" -> 确认删除
```

#### 5. 🎯 截图导入（使用 Snipaste）

快速导入截图内的表情包：

**工作流程：**
1. 使用 Snipaste 截屏
2. 点击 cEmoji 的 "读取剪切板" 按钮
3. 自动识别并保存表情

[📥 下载 Snipaste](https://www.snipaste.com/download.html)
