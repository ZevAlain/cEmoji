# 常见问题

## 支持哪些格式？

图片支持 PNG、JPG、JPEG、GIF。压缩包支持 ZIP。

## GIF 可以复制成动图吗？

可以。cEmoji 会使用内置的 GIF 复制工具保留动图效果。列表中 GIF 会显示角标，悬停 1 秒可预览播放。

## 置顶按钮还会闪烁吗？

不会。旧版切换“始终置顶”时的闪烁问题已经修复，请使用最新版本。

## 为什么导入数量少于选择数量？

通常是因为有重复图片。cEmoji 会按内容去重，重复内容会跳过。

## 快捷键无法注册怎么办？

说明该组合键可能已经被系统或其他软件占用。换一个组合键后重新保存即可。

## 剪贴板导入没有反应怎么办？

先确认剪贴板里确实是图片。可以用 Snipaste、Win+Shift+S 或其他截图工具重新截图后再读取。

## 表情数据在哪里？

源码运行和当前发布包会使用程序目录下的数据：

```text
resource/emoji
resource/emoji_small
etc/pinned.json
```

迁移或备份时，保留这些目录即可。

## 删除后能恢复吗？

不能直接撤销。删除前请确认，重要表情建议先备份。

## 在哪里反馈问题？

请到 [GitHub Issues](https://github.com/ZevAlain/cEmoji/issues) 提交问题，并附上版本号、系统版本和复现步骤。
