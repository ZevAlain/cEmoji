# 常见问题 (FAQ)

## 🔍 常见问题解答

### 安装和启动

#### Q: 应用无法启动，显示错误信息？

**A:** 请按以下步骤尝试：

1. 确保 Windows 已更新到最新版本
2. 以管理员身份运行应用
3. 重启计算机后再试
4. 检查是否有其他应用占用同一端口

#### Q: 从源码安装时缺少依赖？

**A:** 运行以下命令重新安装依赖：

```bash
pip install -r requirements.txt --force-reinstall --no-cache-dir
```

#### Q: 应用提示权限不足？

**A:** 
1. 右键点击 `cEmoji.exe`
2. 选择 "以管理员身份运行"
3. 点击 "是" 确认

### 表情包导入

#### Q: 哪些文件格式支持导入？

**A:** 支持的格式：
- 图片：PNG、JPG、JPEG、BMP、GIF
- 压缩包：ZIP、7z、RAR

#### Q: 如何批量导入表情包？

**A:** 两种方法：

1. **ZIP 导入**（推荐）
   - 将多个表情打包成 ZIP 文件
   - 点击 "导入" -> "导入压缩包"
   - 选择 ZIP 文件自动导入

2. **逐个导入**
   - 点击 "导入" -> "导入图片"
   - 多次选择不同文件

#### Q: 导入的表情包显示不完整？

**A:** 检查：
1. 源文件是否完好无损
2. 文件路径中是否有特殊字符
3. 磁盘空间是否充足

如果上述都无问题，尝试：
```bash
# 清除缓存
rm -rf ~/.cemoji/cache
```

#### Q: 能否从外部硬盘导入表情包？

**A:** 可以。但为了避免读写速度问题，建议：
1. 先将表情复制到本地硬盘
2. 再导入到 cEmoji
3. 或保持外部硬盘始终连接

### 截图功能

#### Q: 读取剪切板功能不工作？

**A:** 
1. 确认已安装 [Snipaste](https://www.snipaste.com/)
2. 检查剪切板中是否有图片
3. 重启应用
4. 以管理员身份运行

#### Q: Snipaste 有其他替代品吗？

**A:** 可以使用任何能将截屏保存到剪贴板的工具：
- Windows 自带的截屏工具（Win+Shift+S）
- 第三方截屏工具（QQ 截屏等）

但 Snipaste 功能最完整，推荐使用。

#### Q: 截图后无法自动识别？

**A:** 检查：
1. 截屏的图片质量
2. 表情包是否清晰
3. 应用是否有剪贴板权限

尝试手动导入代替。

### 性能和存储

#### Q: cEmoji 会占用多少磁盘空间？

**A:** 
- 应用本身：约 50-100 MB
- 表情包数据：取决于导入数量和分辨率
  - 每张 1MB 分辨率表情：约 1-2 MB
  - 1000 张表情：约 1-2 GB

#### Q: 如何释放磁盘空间？

**A:**
1. 删除不需要的表情包
2. 定期清理缓存（设置 -> 清除缓存）
3. 将表情库备份后清空

#### Q: 导入大量表情包时卡顿？

**A:** 
1. 这是正常的，取决于电脑性能
2. 分批导入而不是一次导入全部
3. 关闭其他应用释放内存
4. 重启应用清理内存

### 功能相关

#### Q: 置顶功能为什么会闪屏？

**A:** 这是已知问题，目前无法完全解决。

**临时方案：**
- 使用快捷键替代鼠标点击
- 减少切换置顶的频率
- 等待后续版本更新

#### Q: 能否自定义快捷键？

**A:** 可以。进入 "设置" -> "快捷键"，自定义各功能的快捷键。

#### Q: 支持网络同步表情包吗？

**A:** 目前不支持。这个功能在 v2.1 版本计划中。

#### Q: 能否在多台电脑间同步表情包？

**A:** 目前方法：
1. 定期导出表情包为 ZIP
2. 在另一台电脑导入 ZIP
3. 或手动复制 cEmoji 数据文件夹

### 数据和备份

#### Q: 表情包数据保存在哪里？

**A:** Windows 默认路径：
```
C:\Users\[用户名]\AppData\Local\cEmoji
```

#### Q: 如何备份表情包？

**A:** 三种方法：

1. **文件夹备份**
   - 复制 `AppData\Local\cEmoji` 文件夹

2. **导出为 ZIP**
   - 使用 "导出" 功能

3. **云盘备份**
   - 使用 OneDrive/Google Drive 同步数据文件夹

#### Q: 删除的表情能否恢复？

**A:**
- 刚删除：使用 Ctrl+Z 撤销
- 已保存：从备份文件夹恢复
- 未备份且无撤销：无法恢复

#### Q: 能否迁移到其他电脑？

**A:** 可以。步骤：
1. 在原电脑导出所有表情包
2. 在新电脑安装 cEmoji
3. 导入导出的表情包

或直接复制数据文件夹。
