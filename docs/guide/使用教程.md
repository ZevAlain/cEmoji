# 使用教程

## 📚 详细使用指南

### 基础操作

#### 导入表情包

##### 方法 1：单个图片导入

```
1. 打开 cEmoji 应用
2. 点击顶部菜单栏的 "导入"
3. 选择 "导入图片"
4. 在文件浏览器中选择表情包图片
5. 点击 "打开" 确认
6. 表情包自动添加到库中
```

**支持的格式：** PNG, JPG, JPEG, BMP, GIF

##### 方法 2：批量导入 ZIP 文件

这是最快的导入方式。

**前置要求：** 准备好包含多个表情包的 ZIP 文件

**步骤：**
```
1. 点击 "导入" -> "导入压缩包"
2. 选择 ZIP/7z/RAR 文件
3. 应用自动解压并导入所有表情
4. 完成后显示导入数量统计
```

**ZIP 文件结构示例：**
```
emoji_pack.zip
├── emoji_1.png
├── emoji_2.jpg
├── emoji_3.png
└── subfolder/
    ├── emoji_4.png
    └── emoji_5.png
```

##### 方法 3：截图导入（推荐）

这是最快速的导入方式。

**前置要求：** 安装 [Snipaste](https://www.snipaste.com/)

**步骤：**
```
1. 打开浏览器或其他应用，找到表情包
2. 按 Snipaste 快捷键（默认为 Shift+A）进行截屏
3. 完成截屏后，返回 cEmoji
4. 点击 "读取剪切板" 按钮
5. 截屏中的表情自动保存到库中
```

**Snipaste 快捷键：**
- `Shift+A` - 矩形截屏
- `Shift+B` - 自由截屏
- `Shift+C` - 标题栏截屏

#### 查看和使用表情包

```
1. 在 cEmoji 主窗口中浏览表情包列表
2. 点击想要使用的表情
3. 表情自动复制到剪贴板
4. 在其他应用中按 Ctrl+V 粘贴使用
```

**快速搜索：**
```
在搜索框中输入关键字快速查找表情包
支持模糊搜索
```

#### 删除表情包

**单个删除：**
```
1. 在表情包列表中找到要删除的表情
2. 右键点击该表情
3. 选择 "删除"
4. 在确认对话框中点击 "确定"
5. 表情包被删除
```

**批量删除：**
```
1. 使用 Ctrl+Click 多选要删除的表情
2. 右键点击其中一个
3. 选择 "删除选中项"
4. 确认删除
```

**撤销删除：**
```
Ctrl+Z 撤销最后一次删除操作
```

### 高级功能

#### 窗口置顶

**启用置顶：**
```
1. 点击工具栏中的 "置顶" 按钮
2. 按钮变亮表示已启用
3. cEmoji 窗口将始终显示在其他窗口之上
```

**使用场景：**
- 在视频通话时快速访问表情
- 在写文档时随时查找表情
- 在聊天时保持应用始终可见

**快捷键：**
```
Alt+T 快速切换置顶状态
```

#### 配置快捷键

**访问快捷键设置：**
```
1. 点击 "菜单" -> "设置"
2. 选择 "快捷键" 选项卡
3. 为各个功能配置快捷键
4. 点击 "保存" 应用设置
```

**可配置的快捷键：**
- 显示/隐藏窗口
- 置顶/取消置顶
- 读取剪切板
- 清空搜索

## 🎯 使用场景

### 场景 1：QQ 聊天中快速发送表情

```
1. 打开 QQ 聊天窗口
2. 点击 cEmoji 窗口（或使用快捷键显示）
3. 选择要发送的表情
4. 在 QQ 输入框中按 Ctrl+V 粘贴
5. 点击发送
```

### 场景 2：制作含有特殊表情的文档

```
1. 打开 Word/Google Docs 文档
2. 在需要插入表情的位置放置光标
3. 点击 cEmoji 中的表情
4. Ctrl+V 粘贴到文档
5. 调整大小和位置
```

### 场景 3：整理和备份表情包

```
1. 将各个表情包整理成 ZIP 文件
2. 使用 cEmoji 批量导入
3. 定期备份表情库（复制 cEmoji 数据文件夹）
4. 需要时恢复备份
```

## 🔧 故障排除

### 表情导入后无法显示

**原因：** 图片文件损坏或格式不支持

**解决方案：**
1. 检查原始图片文件是否完好
2. 尝试转换为 PNG 格式后重新导入
3. 查看应用日志了解详细错误信息

### 剪贴板功能不工作

**原因：** 权限不足或系统冲突

**解决方案：**
```bash
# 重启应用
# 或以管理员身份运行
```

### 删除表情后无法恢复

**解决方案：**
- 使用 Ctrl+Z 快速撤销
- 从备份文件夹恢复
- 如果无备份，表情将无法恢复
