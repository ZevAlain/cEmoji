# 使用教程

## 导入

点击“上传”后选择导入方式：

- 上传图片：选择 PNG、JPG、JPEG、GIF 文件。
- 上传 ZIP：批量导入压缩包内的表情。
- 读取剪切板内容：导入当前剪贴板中的图片。

cEmoji 会自动跳过重复图片，并在导入结束后显示统计。

## 复制

在主窗口左键点击表情即可复制。复制后切到目标软件，按 Ctrl+V 粘贴。

GIF 表情会保留动图；如果只是想预览，在 GIF 上停留 1 秒即可播放缩略图。

主窗口上方有“表情包 / 动态表情包”标签。“表情包”显示静态图片，“动态表情包”只显示 GIF。

## 查找和置顶

- 在搜索框输入关键词，可以按文件名筛选。
- 搜索只会筛选当前标签中的表情。
- 右键表情选择“置顶”，常用表情会排在前面。
- 右键置顶表情可以取消置顶。
- 右键表情也可以重命名或打开文件所在位置。

## 删除

- 右键表情选择“删除”可删除单个表情。
- 点击“管理”进入快速删除模式。
- 点击“清空所有表情包”会清空本地表情库。

删除会移除本地文件，重要表情请先备份。

## 快捷呼出

可以设置一个全局快捷键，用于呼出或隐藏主界面。快捷键注册失败时，通常是被其他软件占用，换一个组合键即可。

如果开启“复制后隐藏”，点击表情复制后主窗口会自动隐藏到系统托盘。
