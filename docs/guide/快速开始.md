# 快速开始

## 🎯 5 分钟上手

### 1️⃣ 下载和安装

访问 [GitHub Releases](https://github.com/ZevAlain/cEmoji/releases/latest) 下载最新版本。

### 2️⃣ 启动应用

解压文件后，双击 `cEmoji.exe`（Windows）或对应的可执行文件启动应用。

### 3️⃣ 导入表情包

在应用中选择以下方式之一导入表情包：

#### 方式 A：导入图片文件
```
点击 "导入" -> 选择表情包图片 -> 点击 "导入"
```

#### 方式 B：导入 ZIP 压缩包
```
点击 "导入" -> 选择 ZIP 文件 -> 点击 "导入"
```

#### 方式 C：使用截图导入（推荐）
```
1. 使用 Snipaste 对表情包进行截屏
2. 点击 cEmoji 中的 "读取剪切板" 按钮
3. 截屏自动保存到表情库
```

### 4️⃣ 使用表情包

- 在应用中浏览和选择表情包
- 表情包会被复制到剪贴板
- 在任何应用中粘贴（Ctrl+V）使用

### 5️⃣ 管理表情包

- ✅ 查看已导入的表情包
- ✅ 删除不需要的表情包
- ✅ 窗口置顶功能便于快速访问

## 🎨 使用技巧

### 窗口置顶

点击工具栏的 "置顶" 按钮，保持 cEmoji 窗口始终在最前面，随时随地快速访问表情包。

### 截屏导入最佳实践

1. 使用 [Snipaste](https://www.snipaste.com/) 截屏
2. 按 `Alt+V` 粘贴到 cEmoji（快捷键）
3. 自动识别并保存表情包

## 📌 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Z` | 删除最后一个导入的表情包 |
| `Ctrl+S` | 保存表情包配置 |
| `Alt+Tab` | 切换到 cEmoji 窗口 |

## 🚨 遇到问题？

### 应用无法启动
- 确保 Windows 已更新到最新版本
- 尝试以管理员身份运行应用

### 表情包导入失败
- 确保文件格式正确（PNG、JPG、ZIP）
- 检查文件路径中是否有特殊字符

### 截图功能不工作
- 安装 [Snipaste](https://www.snipaste.com/)
- 检查剪贴板权限

更多问题请查看 [常见问题](/guide/常见问题) 页面。
