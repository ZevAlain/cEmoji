# 快速开始

## 下载和启动

1. 到 [GitHub Releases](https://github.com/ZevAlain/cEmoji/releases/latest) 下载最新版。
2. 解压压缩包。
3. 双击外层 `cEmoji.exe` 启动。

## 导入表情

点击“上传”，可以选择：

- 上传图片：支持 PNG、JPG、JPEG、GIF。
- 上传 ZIP：批量导入 ZIP 内的表情。
- 读取剪切板内容：配合 Snipaste、Win+Shift+S 等截图工具使用。

导入完成后会显示成功、重复跳过和失败数量。

## 使用表情

- 左键点击表情：复制到剪贴板。
- 在聊天、文档或其他软件里按 Ctrl+V 粘贴。
- GIF 表情会保留动图复制能力。
- 鼠标悬停 GIF 1 秒可以预览播放。

## 管理表情

- 搜索框可以筛选表情。
- 右键表情可置顶、取消置顶或删除。
- “管理”模式可快速删除表情。
- “清空所有表情包”会删除本地表情库。

## 窗口

- “始终置顶”可让窗口保持在其他窗口上方，置顶切换闪烁问题已经修复。
- 可以设置全局快捷键，用于呼出或隐藏主界面。
- 最小化会隐藏到系统托盘。
