# 更新日志

## cEmoji V0.0.11

### 更新内容

- 主界面重构为 PySide6 版本，使用更现代的 Qt 组件和模型视图列表。
- 新增“表情包 / 动态表情包”标签，可在静态表情和 GIF 动态表情之间切换。
- 新增 Fluent Glass 风格主题，优化浅色和深色系统主题下的界面显示。
- 新增全局快捷键配置，可呼出或隐藏主界面。
- 新增“复制后隐藏”选项，复制表情后可自动隐藏到托盘。
- 新增“导出所有表情包”功能，可将本地原图表情打包为 ZIP。
- 新增表情排序按钮，支持按时间或名称排序，并可切换升序/降序。
- 新增关于窗口，展示版本、开源许可、运行环境和项目主页入口。

### 体验优化

- 默认窗口调整为三行表情显示高度，界面更紧凑。
- 搜索框会按当前分类显示表情数量，搜索只作用于当前标签。
- 排序按钮使用与标签一致的主题样式，左侧箭头表示升序/降序，右侧文字显示当前按时间或名称排序。
- 表情列表滚动和刷新更流畅，缩略图绘制更稳定。
- GIF 表情保留角标和悬停播放预览，静态表情与动态表情不再默认混在一起。
- 右键菜单支持置顶、取消置顶、重命名、打开文件位置和删除。
- 弹窗按钮统一中文显示，确认弹窗使用“是 / 否”，提示弹窗使用“确定”。

### 修复和清理

- 修复启动时可能出现的 `QFont::setPointSize` 警告。
- 修复关于窗口出现英文 `Show Details` 按钮的问题。
- 修复 GIF 复制后部分软件提示无法加载图像的问题。
- 删除实验性 AI 相关入口、配置和右键菜单。
- 删除不再使用的 `cpgif.exe` 外部复制工具和 `bin` 发布目录复制逻辑。
- 精简默认配置，默认不包含密钥、令牌或本地隐私信息。

## cEmoji V0.0.10

### 更新内容

- 新增右键置顶表情包功能，置顶状态会保存并在下次启动后继续生效。
- 置顶表情包会显示左上角“置顶”角标，GIF 表情继续显示右上角 `GIF` 角标。
- 新增 GIF 悬停播放：鼠标停留在 GIF 表情上一秒后播放缩略图动画，移开后自动停止。
- 新增导入结果统计，导入完成后提示成功导入、已存在跳过和失败数量。
- 图片、ZIP 和剪贴板导入新增内容去重，同图不同名不会重复导入。
- 新增“清空所有表情包”按钮，可一次清空原图、缩略图和置顶记录。

### 修复

- 修复图片和 ZIP 内日文文件名导入后界面显示乱码的问题。
- 修复手绘缩略图后列表项点击区域不正确，导致图片无法点击复制的问题。
- 修复清空表情包提示数量把原图和缩略图重复计数的问题。
- 修复 GIF 复制支持被误移除的问题。
