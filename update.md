# cEmoji V0.0.9

## 更新内容
- 将界面框架从 PyQt5 迁移到 PySide6。
- 重构资源路径、配置读取、表情存储、剪贴板读取等基础模块。
- 图片列表改为 `QListView + QAbstractListModel`，减少大量图片时的控件数量和内存占用。
- 新增目录索引缓存和搜索防抖，提升搜索与刷新速度。
- 图片/ZIP 导入改为后台线程处理，减少导入时界面卡顿。
- 新增全局快捷键设置，可通过快捷键呼出或隐藏主界面。
- 托盘右键菜单调整为“显示界面”和“退出”。
- 最小化默认隐藏到托盘，关闭窗口直接退出。
- GitHub Actions 打包流程改为 Nuitka standalone 目录版，提升发布包启动速度。

## 体验优化
- 修复始终置顶切换时窗口闪烁的问题。
- 优化图片选中态显示，点击图片后立即显示选中效果。
- 管理模式下改为右上角轻量删除按钮，鼠标悬停显示“删除”提示。
- 底部增加“鼠标左键单击图片可复制到剪切板”的操作提示。
- 优化窗口布局，让搜索、操作按钮、快捷键设置和版本信息分区更清晰。

## BUG修复
- 修复源码运行和打包运行时资源路径不一致的问题。
- 修复 ZIP 导入嵌套目录时路径处理错误的问题。
- 修复剪贴板读取可能导致句柄未释放的问题。
- 修复上传按钮重复加入布局的问题。
- 修复旧图片标签实例被全局列表强引用导致的潜在内存泄露问题。
- 修复 PySide6 tooltip 事件中 `QHelpEvent` API 不兼容导致的报错。
