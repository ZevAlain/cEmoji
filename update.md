# cEmoji V0.0.11

## 更新内容
- 主界面重构为 PySide6 版本，使用更现代的 Qt 组件和模型视图列表。
- 新增“表情包 / 动态表情包”标签，可在静态表情和 GIF 动态表情之间切换。
- 新增 Fluent Glass 风格主题，优化浅色、深色系统主题下的窗口、按钮、菜单、标签和列表视觉效果。
- 新增全局快捷键配置，可呼出或隐藏主界面。
- 新增“复制后隐藏”选项，复制表情后可自动隐藏到托盘。
- 新增“更多”菜单，集中放置关于、清空所有表情包等操作。
- 新增关于窗口，展示版本、开源许可、运行环境和项目主页入口。
- 检查更新入口保留在底部，可跳转到 GitHub 最新发布页面。

## 体验优化
- 默认窗口调整为三行表情显示高度，整体界面更紧凑。
- 搜索框会按当前分类显示表情数量，搜索结果只在当前标签内过滤。
- 表情列表改为 `QListView + QAbstractListModel + QStyledItemDelegate`，滚动和刷新更流畅。
- GIF 表情保留角标和悬停播放预览，静态表情与动态表情不再默认混在一起。
- 右键菜单支持置顶、取消置顶、重命名、打开文件位置和删除。
- 置顶功能改用 Windows 原生 `SetWindowPos`，减少切换置顶时的闪烁。
- 弹窗按钮统一中文显示，确认弹窗使用“是 / 否”，提示弹窗使用“确定”。
- 关于窗口移除 Qt 默认的 `Show Details` 按钮，信息直接显示在正文中。

## 配置和隐私
- 精简默认配置，只保留删除状态、全局快捷键和复制后隐藏开关。
- 提交前清理本地配置内容，默认配置不包含密钥、令牌或本地隐私信息。

## BUG修复
- 修复 PySide6 重构后部分导入缺失导致的启动问题。
- 修复全局 QSS 字号设置导致启动时出现 `QFont::setPointSize` 警告的问题。
- 修复关于窗口使用详细信息区域时出现英文 `Show Details` 按钮的问题。
- 修复默认显示所有表情时静态图和 GIF 混在一起的问题。
