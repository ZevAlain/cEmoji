# cEmoji
[![](https://img.shields.io/github/release/ZevAlain/cEmoji.svg)](https://github.com/ZevAlain/cEmoji/releases/latest)
[![LICENSE](https://img.shields.io/github/license/ZevAlain/cEmoji "LICENSE")](./LICENSE "LICENSE")

## 功能
- [x] 导入表情包图片
- [x] 当前窗口始终置顶
- [x] 导入zip格式表情包
- [ ] 本地表情包可选择删除
- [ ] 表情包多标签管理
- [ ] 表情包云服连接
- [ ] 自定义制作表情包
